# Token Template

A minimal starter template for token projects.

This repository provides a lightweight starting point to build and test token-related code, examples, and tooling. Use it as a foundation for ERC-style tokens, token utilities, or experiments.

## Features

- Minimal project structure
- Example token-related placeholders
- Ready for CI / tests

## Getting started

1. Clone the repository:

   git clone https://github.com/nogaenbar/token-template-2.git
   cd token-template-2

2. Inspect the project structure and add your implementation files.
